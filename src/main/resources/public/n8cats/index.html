Моя история знакомства с JetBrains

  • В 2011 году впервые начал писать на Intellij IDEA 10 Flash игру для соц. сетей на языке ActionScript.

  • Затем появилась игра на Unity 3D и C#. Оценил удобство плагина Resharper для Microsoft Visual Studio.

  • Изучал язык Haxe для кроссплатформанной разработки. Помогло что плагин Haxe есть для IDEA.

  • Важное событие для меня когда Google сделали Android Studio 2.0 своей основной IDE в замену Eclipse.
    Тогда я начал активно писать на Java и Kotlin под Android в своих домашних проектах.

  • Появился проект для IOS, написанный на C++.
    Мне на помощь пришёл AppCode!

  • Писал серверную часть на Java в IDEA Community Edition.

  • Великое событие - Google презентавали Kotlin как основной язык для разработки под Android.
    Благо опыта в Kotlin и Android у меня уже хватало и я решил забросить коммерческий GameDev.
    Устроился в ТуТу, они уже использовали Kotlin :)
    Конечно много кода на Java, но всё новое пишем на Kotlin.

  • За все 7 лет знакомства с JetBrains выбираю только ваши продукты.

  • Сейчас разрабатываю домашний проект на 95% написанный на Kotlin.
    Ktor, kotlin-multiplatform, kotlinx-serialization, gradle скрипты на Kotlin.
